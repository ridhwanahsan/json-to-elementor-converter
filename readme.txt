Instantly generate Elementor layouts from AI-generated JSON via a quick editor import button.

== Description ==

JSON to Elementor Converter allows designers and developers to paste Elementor-compatible JSON and instantly turn it into live page layouts.

The plugin includes a dashboard generator page and a quick-import button inside the Elementor editor.

== Features ==

* **Sparkle Button**: Integrates directly into the Elementor "Add New Section" area.
* **Dashboard Generator**: Create a new Elementor page from JSON without opening the editor first.
* **Auto-Normalization**: Automatically wraps partial JSON (widgets/columns) into full sections.
* **Smart IDs**: Generates unique Elementor IDs to prevent editor compatibility issues.
* **History Tracking**: Keeps a background log of your last 10 generations.
* **Lightweight**: Small codebase, focused workflow, and professional folder structure.

== Installation ==

1. Upload the `json-to-elementor-converter` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Open any page with Elementor and look for the sparkle icon in the Add Section area.
